# mini-rum

打造一个自己日常工作中使用的 vue 组件库, 拒绝复制粘贴

## 项目运行

```
cnpm i & npm run dev & 见 package.json
```

## install(安装)

> npm instal mini-rum -S or yarn add mini-rum -D

## Quick Start

```
import Vue from 'vue'
import Minirum from 'mini-rum'

Vue.use(Minirum)

// or
import {
  Button
  // ...
} from 'mini-rum'

Vue.component('m-button', Button)
```
